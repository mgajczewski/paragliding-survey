### Documentation for the Ballast Fairness Survey

This folder contains the source material and aggregated resources that will be used to create the Ballast Fairness Survey. It includes background reading, discussion archives, and reference documents intended to provide a comprehensive basis for the survey's questions and content.

### Contents

- **Knowledge**: The `knowledge` subfolder contains extracted reference material and context for this debate and survey, including background, proposals, and rationale. The knowledge pages were copied from GaggleReport body content, forming the most important and reliable source for the survey.
